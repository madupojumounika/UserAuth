# UserAuth System 🔐

A simple full-stack user authentication system built using React, Node.js, Express, and PostgreSQL.

## 🚀 Features
- User Signup
- User Login
- Password Encryption (bcrypt)
- Dashboard after login
- Responsive UI

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- Security: bcryptjs

## 📂 Project Structure
UserAuth/
│── backend/
│── frontend/

## ▶️ How to Run

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
